Lunar Wing
Author: Dmitry Nikolaevich Demidov


Lunar Wing is a Discord bot. All commands are listed below. Prefix is "+".

+help/+h/+commands/cmmds - Shows all available commands.
+rushelp/+ruscommands/+ruscmmds - Shows Russian help page.
+randnum - Generates a random number to 10000.
+admshell/+admshell - Admin command shell.
+dev-terminal/+shell/+terminal - Developer commands.
+hug [@user] - Hugs the user.
+kiss [@user] - Kisses the user.
+gulag [@user] - Send someone to gulag! (just kidding)
+tb64 [word] - Converts your text to Base64.
+fb64 - Converts your text from Base64.
+ask [something] - Probably the useless command.
+history - Shows the bot updates.
+about - Shows an info about the bot.
+userinfo - Information about yourself.
+reverse [word] - Reverse to esreveR
+say [word] - Say the word.
+vapor [text] - V a p o r . t e x t
+invite - Bot invite link
+fight [@user] - Fight with someone!
+duck [@user] - You are duck! (the funny command)
+serverinfo - Information about server.
+delinc [includes channel-name] - Delete channels that include that name (Admin only)
+random - Random quote
+vs/+2vs2 - Random battles
+time - System time
+runtime - Shows how long bot works.
+eval [js code] - Evaluates a JS code [Very dangerous, developer only]
+urban [word] - Shows the Urban Dictionary definition for a specified word/term.
